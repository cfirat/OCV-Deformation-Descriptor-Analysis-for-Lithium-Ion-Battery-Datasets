# Candidate Portable Geometric Descriptors of Lithium-Ion OCV Deformation

This repository contains processed descriptor tables, analysis notebooks, and figure-generation scripts for the manuscript:

"Candidate Portable Geometric Descriptors of Lithium-Ion OCV Deformation Across Aging- and Temperature-Resolved Battery Datasets"

## Data sources
- Oxford Battery Degradation Dataset 1: [DOI ...](https://doi.org/10.5287/BODLEIAN:KO2KDMYGG)
- CALCE A123 battery data: [URL ...](https://calce.umd.edu/battery-data)

## Reproducibility
Run notebooks in order:
1. 01_oxford_download_parse_pseudo_ocv.ipynb
2. 02_oxford_descriptors_gp_uncertainty.ipynb
3. 03_calce_a123_descriptors_gp_uncertainty.ipynb
4. 04_cross_dataset_comparison.ipynb
5. 05_paper_final_results_audit_and_figures.ipynb

The final manuscript figures and tables are generated by notebook 05.

## Key descriptor
D6 is reported as NDIR_RMS, the RMS voltage-range-normalized deformation index.
